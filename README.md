# 🚛 Nomad Base: The Last Trucker

**노마드 베이스**는 아포칼립스 세계관을 배경으로 한 방치형 전략 생존 게임입니다. 당신의 기지이자 이동 수단인 트럭을 업그레이드하고, 다양한 대원들을 영입하여 최후의 낙원을 찾아 떠나세요!

## ✨ 주요 특징
- **실시간 탐사**: 트럭이 이동하며 자동으로 고철과 식재료를 수집합니다.
- **다양한 대원**: Common부터 Super Rare까지 20종의 대원들이 각기 다른 보너스를 제공합니다.
- **무한 요리**: 획득한 재료로 12종 이상의 풍성한 특별 요리와 수천 가지의 조합 요리를 발견할 수 있습니다.
- **지역 보스**: 위험 구역을 지키는 보스들과 전투를 치르고 새로운 지역을 해금하세요.
- **모바일 최적화**: PC뿐만 아니라 모바일 브라우저에서도 완벽하게 구동됩니다.

## 🛠️ 기술 스택
- Vanilla JavaScript (No Framework)
- CSS3 (Custom Properties & Animations)
- HTML5 (LocalStorage Persistence)
